# 🌱 AgroSense Dashboard

A modern React dashboard for monitoring smart agriculture sensor data in real time.

## 🚀 Features
- Reusable `SensorCard` components
- Soil Moisture monitoring
- Temperature tracking
- Humidity monitoring
- Animated progress bars
- Responsive flexbox layout
- Dark agricultural-themed UI

## 📂 Project Structure
```bash
agrosense-dashboard/
├── index.html
├── package.json
├── vite.config.js
└── src/
    ├── main.jsx
    ├── App.jsx
    ├── SensorCard.jsx
    └── index.css
```

## 🛠️ Technologies Used
- React 18
- Vite
- CSS Flexbox

## ▶️ How to Run
```bash
npm install
npm run dev
```

Open:
```bash
http://localhost:5173
```
